# Estudo de Diferentes Métodos de Imputação para Dados Faltantes

## Descrição

Este projeto de iniciação científica voluntária, orientado pela Profª Lourdes Coral Contreras Montenegro (Estatística – ICEx/UFMG), tem como foco o estudo teórico e computacional de diferentes métodos de imputação para dados faltantes.

A pesquisa aborda a problemática da falta de observações em conjuntos de dados, investigando técnicas estatísticas para tratar esse fenômeno de forma adequada, preservando o tamanho amostral e evitando vieses nas análises subsequentes.

------------------------------------------------------------------------

## Objetivo

Este projeto tem como principais objetivos:

- Possibilitar o estudo da metodologia estatística aplicada à falta de observações em conjuntos de dados;
- Proporcionar ganho de maturidade estatística e computacional por meio do aprendizado de diferentes técnicas de imputação.

------------------------------------------------------------------------

## Justificativa

Em diferentes áreas do conhecimento, a coleta de dados é fundamental para a análise e conclusões de uma pesquisa. No entanto, a falta de observações em um conjunto de dados pode:

- Reduzir o tamanho da amostra;
- Produzir resultados não esperados ou enviesados.

A imputação de dados com o método adequado pode solucionar esses problemas, preservando o tamanho amostral original e permitindo decisões mais precisas. Assim, o estudo teórico e computacional dos métodos de imputação é essencial para qualquer pesquisa a ser desenvolvida.
